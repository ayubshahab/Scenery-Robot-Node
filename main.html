To get Started:

  • Click the HOME icon from the right panel and enter the Width and Depth of your stage in feet.
  • Select any additional inches from the drop down options.
  • Adjust the Interval Rate (for points extraction) using the range slider.
  • Then click the SUBMIT button. You can now begin drawing.

Understanding the path drawn

When you draw something on screen, the program automatically starts extracting critical points (locations your mouse has been) based on a default interval rate. The dault rate is set to .05 second, as shown in the following image, and can be adjusted using the range slider. An increase in the interval rate results in a decrease in the number of critical points that are extracted. When modifying the range slider, it is important to press the SUBMIT button again as this is what updates the program with any changes that may have been made. These critical points are essential to the program as they are converted into feet and inches and sent to the robot when such a request is made by the user.

Viewing Points Extracted & Line Drawn

To view the points that have been extracted, use the toggle switch under the PENCIL icon labeled as "Critical Points". To toggle and untoggle the drawn line, use the switch labeled as "User Drawing".

The Grid Function

The grid function of this program relies on the dimensions of the stage inputted as well as the interval rate selected from the drop down from inside the Grid Tab.

To toggle the grid feature:

  • Click the HOME Icon and enter the respective dimensions of the stage, then click SUBMIT.
  • Select the GRID Icon and choose the interval rate from the drop down
  • Press the Grid Display Switch to turn on grid display
  • To change the Grid Interval Display, simply select a new Grid Interval rate from the drop down and re-toggle the Grid Display Switch.

Canvas Layers

This program utilizes various layers of canvas' to provide as much information as possible.

  • The Grid Canvas displays the grid intervals and can be toggled On and Off under the GRID Icon.
  • The Ground Plan Canvas displays the user uploaded ground plan of the stage and can be toggled On and Off under the GROUND PLAN Icon.
  • The User Drawing Canvas sits on top of the Ground Plan Canvas and Grid Canvas and this layer is where the user draws the desired path. This Canvas can be toggled On and OFf under the PENCIL Icon.
  • The Critical Points/ Edit Canvas sits on top of the User Drawing Canvas and this is used to display the critical points that the program is keeping track off as well as allow the user to make modifications to the track via the Erase Function. Both the toggle switch for this canvas and the Erase Function can be found under the PENCIL Icon.

Note: The "Critical Points" toggled switch, under the PENCIL icon, toggles the Critical Points/ Edit Canvas, however, while the Points Canvas is enabled, the user cannot draw additional paths. The Points Canvas must be disabled to draw paths and can only be used to erase/ edit the path drawn.

Note: When you start using this program, you must maintian the same window size from start to finish, or anytime you may come back to continue working on a program you were working on previously. The recommended size is just to maximize the window screen as this alleviates any potential problems.

How To

The Drawing Function

We have already briefly coverd the drawing function up above. Simply, select the HOME icon, enter the respective dimensions of the stage, and click the SUBMIT button. You should be able to draw paths anywhere on the screen, however there are some things to pay attention to when drawing.

  • Once a path has been drawn, the only way to continue drawing is to either draw at the beginning of the path or the end.
  • Since the robot cannot teleport around stage, there must be a single continuous path at all times.
  • When you are done drawing a path, you can select the PENCIL icon and toggle the "Critical Points" switch to visualize the critical points captured by the program.
  • When you use the edit feature in the middle of a path, the next line can only be drawn starting from one of the edit locations, indicated by a red dot, and must connect with the other edit location.

The Erase Function

The erase function is used to edit in the middle of paths as well as erase entire paths drawn. To erase, simply toggle the "Critical Points/ Edit Canvas" switch under the PENCIL icon which will highlight all the critical points that are stored in the program. Then you can select any two points and click the erase button. There are four such cases of using the Erase Function.

  • The first case is when you select somewhere in the middle of the path and the last point in the path. In this case, the user can start drawing from the middle point that was selected as this becomes the new end point.
  • The second case is when you select somewhere in the middle of the path and the first point in the path. In this case, the user can also start drawing from the middle point as this becomes the new start point.
  • The third case is when you select two points that are in the middle of the path drawn. In this case, using the erase function results in only being able to draw from one of the middle edit points. You must first complete the path between these two points before you can start drawing elsewehere.
  • The last case is when the first and last points in the path are selected. In this case, the user can start drawing anywhere on the canvas after pressing the "Erase" button.

Saving your Workspace

After you are done drawing your paths, select the CLOUD icon and press the SAVE button and the program will allow you to save your workspace to your desired location. NOTE: the dimensions (size) of your browser window must be the exact same when you upload and start working again.

Uploading your Workspace

To begin working where you left off, you must first resize your browser window to the dimensions that you were using before you saved your workspace. To upload your workspace, simply select the CLOUD icon and click the UPLOAD WORKSPACE button and the program will allow you to select the desired file.

Get Points

The "Get Points" button, under the PENCIL icon, allows you to see the coordinates of the critical points stored in the program converted into feet/inches.

Reset

The "Reset" button, under the PENCIL icon, allows you to reset the program where all the stored data is discarded.

Copy Points to Clipboard

The "Copy Points to Clipboard" button, under the PENCIL icon, is utilized after the "Get Points" button is pressed and coordinates (in feet/inches) are obtained. Allows for easier copy/paste functionality.

Send Path to Robot

After you are done drawing a path, the set of critical points stored in the program can be sent to the robot via the SEND PATH TO ROBOT icon which automatically converts the coordinates into feet/inches.
